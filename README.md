# Student Management web 📱

Application web et mobile de gestion des étudiants développée avec React Native & Expo.

## 🚀 Technologies utilisées
- Laravel
- React
- Inertia js
- TypeScript

## 📋 Prérequis
- Node.js >= 18

## ⚙️ Installation

# Aller dans le dossier
cd student-management

# Installer les dépendances
npm install

## 🎯 Lancer le projet
npm run dev

## 📱 Plateformes supportées
- ✅ web

# Student Management web et Mobile 📱

Application web et mobile de gestion des étudiants développée avec React Native & Expo.

## 🚀 Technologies utilisées
- React Native
- Expo
- Node.js
- Laravel
- React

## 📋 Prérequis
- Node.js >= 18
- Expo CLI
- Android Studio (pour l'émulateur Android)
- Xcode (pour le simulateur iOS)

## ⚙️ Installation

# Cloner le projet
git clone https://github.com/ton-username/student-management-mobile.git

# Aller dans le dossier
cd student-management-mobile

# Installer les dépendances
npm install

## 🎯 Lancer le projet
npx expo start

## 📱 Plateformes supportées
- ✅ Android
- ✅ iOS
